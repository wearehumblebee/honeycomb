# Grid

Grid is a container for css grid layouts

https://theme-ui.com/components/grid

<Meta title="UI Components / Layout / Grid" parameters={{ layout: 'centered' }} />

You can define variants in your theme under `theme.grids`

https://developer.mozilla.org/docs/Web/CSS/grid-template-columns

https://theme-ui.com/recipes/css-grid

You can use arrays of values to change those between the different breakpoints of your theme, defined under `theme.breakpoints`

The first value will be for mobile

https://theme-ui.com/getting-started/#responsive-styles
