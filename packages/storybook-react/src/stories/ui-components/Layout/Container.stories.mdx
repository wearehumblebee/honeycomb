# Container

Container is a wrapper with a max-width centering its content.

https://theme-ui.com/components/container

<Meta
  title="UI Components / Layout / Container"
  component={Container}
  parameters={{ layout: 'centered' }}
/>

<Props of={Container} />

You can define variants in your theme under `theme.layout`, default variant is `container`

You can define also define a max-width under `theme.sizes.container`
