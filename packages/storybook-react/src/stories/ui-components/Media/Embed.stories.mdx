# Embed

Embed is a component rendering responsive iframes (videos, pages, anything) with a specific ratio

https://theme-ui.com/components/embed

<Meta title="UI Components / Media / Embed" parameters={{ layout: 'centered' }} />

NOTE: to optimise media performances, implement a lazyloading strategy:

https://developers.google.com/web/fundamentals/performance/lazy-loading-guidance/images-and-video
