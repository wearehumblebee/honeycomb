# Card

Card is a component rendering styled metadata or category tags

https://theme-ui.com/components/card

<Meta title="UI Components / Content / Card" component={Card} parameters={{ layout: 'centered' }} />

<Props of={Card} />

You can define variants in your theme under `theme.cards`, default variant is `primary`

<Preview>
  <Story name="Default">
    {() => {
      const { theme } = useThemeUI();
      return (
        <Card variant={select('Variant', theme ? Object.keys(theme.cards || {}) : [], 'primary')}>
